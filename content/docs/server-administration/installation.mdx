## 1.1. System Requirements

QuasarDB runs on several architectures, and requirements differ depending on which architecture you're targeting.

### 1.1.1. x64 build

- An Intel Haswell or better microarchitecture
- End-to-end ECC memory highly recommended
- At least 512 MiB of RAM
- At least 1 GiB of disk

### 1.1.2. Core2 build

The Core2 build is optimized for older CPUs, at a slight cost of performance.

<Callout type="note" title="Note">
  ARM64 builds may need special tuning to give you the best experience. Contact your Solution Architect for configuration recommendations.
</Callout>
