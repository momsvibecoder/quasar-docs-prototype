QuasarDB exposes a SQL-like query language optimized for analytical work over time-series and numerical data.

```sql
SELECT avg(close)
FROM stocks
RANGE '2026-05-11T09:00:00Z' TO '2026-05-11T10:00:00Z'
GROUP BY 1m
```
